## Things used

- NodeMCU ESP8266
- Jumper wires
- Li-Po battery (3S 11.1 V)
- DC-DC voltage controller
- L298N Motor Controller
- HC-SR04 Ultrasonic sensor
- Buzzer
- LEDs (red & green)
- DC Motor (w/ toy car)
- Toy Car with Motor
